# Credit score classification
Этот проект решает задачу кредитного скоринга для мультикласса. 

Цель проекта: с помощью алгоритмов машинного обучения построить модель, которая сможет определять кредитный рейтинг людей по заданным признакам.
Финальная модель строилась по методу градиентного бустинга.
## Данные
Данные взяты с платформы Kaggle https://www.kaggle.com/datasets/parisrohan/credit-score-classification/data<br>

Датасет содержит 100000 объектов и 28 признаков. Пример признаков:<br>
"Age"<br>
"SSN"<br>
"Occupation"<br>
"Annual_Income"<br>

В качестве таргетов используется 3 класса:<br>
"Poor"<br>
"Standard"<br>
"Good"<br>

## Структура проекта

- Исправил ошибки в исходных данных, ограничил допустимые значения признаков, проанализировал распределения признаков и возможных выбросов.
- Построил preprocessing pipeline с заполнением отсутствующих значений и кодировкой категориальных признаков.
- Построил baseline модели: Logistic Regression и Random Forest.
- Выполнил оптимизацию гиперпараметров модели XGBoost с помощью RandomizedSearchCV.

## Основной результат
ROC AUC (macro) = 0.85<br>
Average Precision (macro) = 0.717
